BILLBOARD HIT SONGS PREDICTION APPLICATION

In the project deployed via Django, there are two options: "Upload Song" and "Spotify Link". "Upload Song" allows users to upload songs from their local host, and the application predicts whether the song is popular or not based on machine learning algorithms. The selection of the best model is based on accuracy. "Spotify Link" allows users to input a link from the Spotify website directly. Using machine learning algorithms, the application predicts whether the song associated with the provided link is popular or not
